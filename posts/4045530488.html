变量定义

变量一旦定义就有默认值

运行结果：

0,“”

变量赋初值

运行结果：

1 3 Hello Go!

自动检测变量类型

运行结果：

1 3 true def

1 5 true def

包变量

运行结果：

11 def true

内建变量

  • bool, string

  • (u)int, (u)int8, (u)int16, (u)int32, (u)int64, uintptr

  • byte(8bit), rune(32bit)

  • float32, float64, complex64, complex128

    • complex小练习：验证欧拉公式 eπi+1=0e^{πi} + 1=0eπi+1=0

    • 运行结果：

      (0.000+0.000i)

类型强制转换

运行结果：

5

常量的定义

运行结果：

abc.txt efg.txt 5
